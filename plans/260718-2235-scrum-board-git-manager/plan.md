## Overview
Web App quản lý dự án tối giản theo quy trình Scrum/Sprints, tích hợp tự động hóa qua GitHub Webhooks với giao diện tối (Dark Mode) Slate lạnh (phong cách qa-qtdn) và Realtime updates.

## Tech Stack
* Frontend: React 19, TypeScript, Vite 7, TailwindCSS v3.
* Router & State: React Router Dom v6, @tanstack/react-query v5.
* Database & Auth & Realtime: Supabase (PostgreSQL, Realtime).
* Drag and Drop: @dnd-kit/core.
* Charts: Recharts.
